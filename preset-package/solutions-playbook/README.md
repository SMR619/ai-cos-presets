# Solutions Playbook

**Owner:** Ava (maintained) / Steve (reviews)
**Last Updated:** 2026-03-31

## What This Is

A living reference library of vetted tools, processes, and systems we've evaluated and chosen for running Steve's businesses and the AI Chief of Staff product. Every entry documents the problem, the solution, the cost, why we chose it over alternatives, and whether Ava can use it via API.

## Two Layers

**Layer 1 — Internal (us):**
Before recommending any tool for a new project, Ava checks PLAYBOOK.md first. If the problem is already solved, we use the existing solution. No re-evaluating from scratch.

**Layer 2 — Product (AI CoS clients):**
This becomes the "Operator Toolkit" module — a curated add-on that tells clients: here's the vetted solution for X problem, here's why, here's what it costs, and your agent already knows how to use it. That's a meaningful differentiator over any blank-slate AI tool.

## How to Use

- **New project:** Check PLAYBOOK.md before recommending tools
- **New solution landed:** Add entry before the session ends (while reasoning is fresh)
- **Stale entry:** Tag with 🔴 REVIEW NEEDED, surface at next Weekly CEO-COO Review
- **Review cadence:** See REVIEW-PROTOCOL.md
