# Deal Knowledge

Document the types of deals you do, deal structures you use, red flags you watch for, and lessons you've learned.

## Format

For each deal type/structure, include:
- How it works
- Red flags (what to watch for)
- Green lights (what you want to see)
- An example deal
- Lessons learned

## Template

```markdown
## Deal Structure: [Structure Name]

**How it works:** [Explain the mechanics of this deal structure - what are the steps, who's involved, what's the outcome?]

**Red flags:** [What to watch for - things that indicate a bad deal or high risk]

**Green lights:** [What makes this a good deal - positive indicators, what you want to see]

**Example:** [A real deal you've done using this structure - what happened, what you learned]

**Lessons learned:** [Key insights from doing this deal type repeatedly]

**Typical ROI/Returns:** [What returns you expect from this structure]

**Typical timeline:** [How long from start to finish?]
```
