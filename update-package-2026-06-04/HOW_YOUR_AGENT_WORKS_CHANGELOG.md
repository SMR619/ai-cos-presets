# How Your Agent Works — Changelog

## v1.0 — 2026-06-04 (Initial Release)

**What changed:**
Created comprehensive foundation module explaining the 5 core systems that make the agent work.

**Components:**
- Memory Architecture (remembers your business context)
- Operating Rules (decision-making framework)
- Framework Library (proven thinking frameworks)
- Tools Knowledge Base (database of solutions)
- Knowledge Base System (learning from experience)

**Why:**
Customers need to understand what makes the agent competent. Without understanding the foundation, they might not appreciate why the agent works well. This module is the first thing new agents should read.

**Key insight:**
These 5 systems are not separate features. They work together to create a competent, learning agent. Every other module is powered by these systems.

**Status:**
Live in all presets. New agents read this first.
